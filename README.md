# 松元 貴光 / Takamitsu Matsumoto

## プロフィール

業務改善・社内SEの実務経験を活かし、Python / ExcelVBA / 生成AIを活用した業務効率化ツールを開発しています。

これまでの現場経験をもとに、議事録作成、売上集計、Excel業務の自動化、OCR読取、社内ナレッジ検索など、実務で発生しやすい課題を技術で解決することを重視しています。

## 開発しているもの

### [AI議事録自動生成アプリNexusMinute](https://github.com/tm-business-ai/nexusminute)

会議後の議事録整理や共有作業を効率化するためのAI議事録作成アプリです。  
音声ファイル、ブラウザ録音、会議メモ、文字起こし済みテキストをもとに、要点整理、決定事項、次の作業、共有用まとめを生成します。

### [社内ナレッジ検索AI](https://github.com/tm-business-ai/enterprise-knowledge-hub-ai)

社内マニュアルやFAQ、規程、議事録などの文書を登録し、AIで検索・要約できるRAG型の業務改善アプリです。  
質問に対して、登録文書をもとに根拠つきで回答し、社内問い合わせ対応や資料確認の効率化を目的にしています。

### [売上データ自動集計・月末売上管理ツール](https://github.com/tm-business-ai/sales-report-manager)

CSV・Excelの売上データを取り込み、月末確認用のExcelレポートを自動作成する業務改善アプリです。  
商品別・カテゴリ別・日別・月別の集計、月末サマリー、前月比較、未分類データやエラー行の確認を支援します。  
毎月発生する集計作業や確認作業を効率化し、作業時間の削減とミスの防止を目的にしています。

### [月末売上管理ツール](https://github.com/tm-business-ai/monthly-sales-report-tool)

CSVデータを取り込み、月次の売上集計やExcelレポート出力を支援する業務改善アプリです。  
毎月発生する集計作業や確認作業を効率化し、作業時間の削減とミスの防止を目的にしています。

### ExcelVBA業務自動化サンプル

転記、集計、帳票作成など、Excelで発生しやすい定型業務の効率化を想定したサンプルです。  
現場担当者が使いやすい操作性と、業務フローに合わせた設計を意識しています。

### [請求書・領収書OCR読取＋消込支援ツール](https://github.com/tm-business-ai/invoice-ocr-reconciliation)

請求書・領収書のOCR読取結果と台帳データを照合し、消込候補や未一致データの確認を支援する業務改善アプリです。  
経理・事務作業で発生しやすい確認作業や転記作業を減らすことを目的にしています。

### [LINE予約受付・予約管理システム ReserveLine Pro](https://github.com/tm-business-ai/reserveline-pro)

美容室、整体院、個人レッスン、士業相談などの小規模事業者向けに、LINEでの予約受付・確認・キャンセルを想定した予約管理システムです。  
店舗側は管理画面から、本日の予約、今週の予約、電話・店頭予約の登録、キャンセル管理、予約メニュー設定、予約データのダウンロードを確認できます。  
FastAPI、Streamlit、SQLiteを使用し、LINE公式アカウントと接続する前でも動作確認できる構成にしています。

## 開発で意識していること

業務改善・社内SEの経験をもとに、単にコードを書くことだけではなく、実際の業務で使いやすい形にすることを意識しています。

特に、以下の点を大切にしています。

- どの作業に時間がかかっているのか
- どこで二重入力や確認作業が発生しているのか
- どの部分を自動化すれば現場の負担が減るのか
- 現場担当者が迷わず使える画面・操作手順になっているか
- ExcelやCSVなど、実務で使われているデータ形式に対応しているか
- 入力ミスや確認漏れを見つけやすい構成になっているか
- READMEや操作マニュアルを整備し、他の人でも使える状態になっているか
- サンプルデータは架空データを使用し、実データや個人情報を含めていないか
- テストを用意し、主要機能が壊れていないか確認できるか

現場の課題を整理し、Python / ExcelVBA / 生成AIを活用して、実務に役立つ形へ落とし込むことを目指しています。
